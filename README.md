# firstchatbot
repository for first chatbot with rasa deployed on heroku

## How to use

Click on the button below to deploy this template on your Heroku instance.
Heroku will automatically build the Docker image and your project's NLU model.

[![Deploy to Heroku](https://camo.githubusercontent.com/83b0e95b38892b49184e07ad572c94c8038323fb/68747470733a2f2f7777772e6865726f6b7563646e2e636f6d2f6465706c6f792f627574746f6e2e737667)](https://heroku.com/deploy?template=https://github.com/MIRIBO4630/firstchatbot)

_It takes a couple of minutes to build and start the server. You can see the progress in Heroku logs._
